Python has two different types of parameters:

1.
Positional parameters

Each argument in a call has a matching parameter in the function or method definition.

2.
Keyword parameters

Allows to specify a default value.

A keyword parameter is written as a variable name, an equal sign, and a default value.

def someFunction(<keywordParameter>=<default value>):

There can be multiple keyword parameters, each with a name and a default value.



Any function or method can have both positional parameters ad keyword parameters.

All positional parameters must be specified before any keyword parameters.

def someOtherFunction(positionalParam1, positionalParam2, <keywordParameter1>=<defauilt value1>, <keyWordParameter2>=<default value 2>):



Example:

def orderBurgers(nBurgers, kethcup, mustard, pickles):

orderBurgers(2)

TypeError: orderBurgers() missing 3 required positional arguments: 'ketchup', 'mustard', and 'pickles'



Rewritten:

def orderBurgers(nBurgers, ketchup=True, mustard=True, pickles=True):

orderBurgers(2) # 2 burgers with all condiments

orderBurgers(2, mustard=False, pickles=False) # 2 burgers with ketchup

orderBurgers(2, True, False, False) # 2 burgers with ketchup



Positional parameters must come before any optional keyword parameters.

Order matters for positional parameters, keyword parameters can be placed in any order.



Example:

orderBurgers(2, mustard=False, pickles=False) # 2 burgers with ketchup

orderBurgers(2, pickles=False, mustard=False, kethcup=False) # 2 burgers plain



A keyword paramaeter can have a default value of any data type.


Example:

def orderIceCream(flavor, nScoops=1, coneOrCup='cone', sprinkles=False):

The values can be overwritten when making the function call.



<keyWordParamater>=<default value> or <keyWordParamater> = <default value> both can be used, but the first is standard.



The downside to using default values for functions and methods is that when the codebase gets large, it can be difficult to change the keyword name.

It is very important to use default values that should suit thee widest possible range of users.
